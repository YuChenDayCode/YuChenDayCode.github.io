 # 珍藏！！ JS CSS不传之秘集合

```javascript
<!-- 其中5指每隔5秒刷新一次页面 -->
<meta http-equiv="refresh" content="5"> 
================================================================================
animation:动画播放
================================================================================
图片模态：https://c.runoob.com/codedemo/157
================================================================================
p:before{ content:"Read this -";background-color:yellow;color:red;font-weight:bold;} //前后加固定的东西
p:after{ content:"- After this";background-color:darkgray;color:red;font-weight:bold;}
================================================================================
p.a{text-transform:uppercase;}全部变大写
p.b{text-transform:lowercase;}全部变小写
p.c{text-transform:capitalize;}首字母变大写
================================================================================
<p>X<sup>2</sup>+log<sub>e</sub>e</p><br/><br/>  X2+logee
================================================================================
fadeToggle():重载，可以传时间 之类的
================================================================================
Autocomplete:输入模糊匹配候选
================================================================================
.nowrap{white-space:nowrap;}/* 强制不换行 */
.breakword{word-wrap: break-word;}/* 允许单词内断句，首先会尝试挪到下一行，看看下一行的宽度够不够，不够的话就进行单词内的断句 */
.breakAll{word-break:break-all;}      /* 断句时，不会把长单词挪到下一行，而是直接进行单词内的断句 */      
.ellipsis{text-overflow:ellipsis;overflow:hidden;}/* 超出部分显示省略号 */
================================================================================
div[id='1'] p：后代选择
div[id='2']>p：子类选择
div[id='3']+p：相邻兄弟(后面第一个)
div[id='4']~p：同级选择（后面的）
.parent() 父元素
.children() 子元素
.prev() 前一个同辈元素
.next() 下一个同辈元素
================================================================================
^=：开头
$=：结束
*=：包含
================================================================================
//评价星星
 if(rating > 5 || rating < 0) throw new Error('数字不在范围内');

//动态创建
var s = document.createElement('script');
s.setAttribute('src', '@require');
document.body.appendChild(s)
================================================================================
autocomplete="off"  //不保留历史输入文本框
================================================================================
navigator.userAgent.toLowerCase() 获得浏览器系统信息
===============================================================================
window.history.go(-1);//返回上一页并刷新 但很多时候不会刷新
window.location.href = document.referrer;//返回上一页并刷新  document.referrer 获取上一个页面的地址
```
